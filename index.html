INSTRUCCIONES

🎯 Objetivo

Protege los tres pilares de seguridad (CID) de los ataques cibernéticos:

  • 🔵 Confidencialidad: Protege datos sensibles
  • 🟢 Integridad: Previene modificaciones no autorizadas
  • 🔴 Disponibilidad: Mantiene servicios accesibles

🎮 Cómo Jugar

  • Responde preguntas correctamente para construir defensas
  • Cada respuesta correcta = Nueva torre disponible
  • Cada error = Enemigos más fuertes
  • Usa las defensas adecuadas contra cada tipo de amenaza
  • Mira tus estadísticas de respuestas en tiempo real

📊 Sistema de Puntuación

  • ✅ Respuesta correcta: +50 puntos
  • ❌ Respuesta incorrecta: -salud del defensor
  • 🎯 La precisión afecta bonificaciones

🛡️ Defensas Disponibles

  • Firewall: Defensa básica contra virus
  • Cifrado: Protege contra fugas de datos
  • IDS: Detecta y elimina amenazas avanzadas
  • Antivirus: Fuego rápido contra amenazas

🎯 Controles

  • Click: Construir torre seleccionada
  • Teclas 1-4: Seleccionar torres
  • ESC: Deseleccionar torre
  • ESPACIO: Alternar panel de torres
